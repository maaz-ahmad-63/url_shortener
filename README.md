URL Shortener – Node.js

A simple and efficient URL Shortener built with Node.js, Express, and MongoDB that converts long URLs into short, shareable links. This project demonstrates backend development concepts such as REST APIs, database integration, routing, and URL redirection.

⸻

🚀 Features
	•	🔗 Convert long URLs into short URLs
	•	📊 Track number of clicks for each shortened link
	•	↪️ Automatic redirection to the original URL
	•	🗂 Store URLs in MongoDB database
	•	⚡ Fast and lightweight REST API
	•	🔐 Basic validation for URLs

⸻

🛠 Tech Stack
	•	Backend: Node.js, Express.js
	•	Database: MongoDB
	•	ODM: Mongoose
	•	Utilities: nanoid / shortid (for generating short codes)
